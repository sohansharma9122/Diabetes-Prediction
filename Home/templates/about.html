About Diabetes Prediction Project

This project provides an easy-to-use online platform where patients can check if they are at risk of diabetes. Users can either fill their health data manually (like glucose, BMI, age, pedigree, insulin, etc.) or upload a medical report photo. The system automatically extracts the data and predicts whether the patient has diabetes.

Features

  • User-friendly interface suitable for both patients and doctors.
  • Accurate predictions using advanced Machine Learning and Deep Learning models.
  • Data collection via manual input or report upload (automatic data extraction).
  • Location-based suggestions for the nearest diabetes doctors if the prediction is positive.
  • All data securely recorded in the admin panel.

Technology Stack

  • Backend: Python, Django
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Machine Learning & Deep Learning: Predictive algorithms for diabetes detection
  • Computer Vision: OpenCV for automatic report data extraction
